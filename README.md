# Порівняльний аналіз результатів обчислення інтеграла

## 1. Аналітичний розрахунок
Математичне значення визначеного інтеграла функції $f(x) = x^2$ на проміжку $[0, 2]$ обчислюється наступним чином:

$$
\int_{0}^{2} x^2 \,dx = \left[ \frac{x^3}{3} \right]_{0}^{2} = \frac{2^3}{3} - 0 = \frac{8}{3} \approx 2.666667
$$

Перевірка за допомогою функції `scipy.integrate.quad` дала ідентичний результат: `2.666666666666667`.

## 2. Результат методу Монте-Карло
Під час виконання програмної реалізації алгоритму з генерацією **N = 100,000** випадкових точок було отримано наступний результат:
* **Інтеграл (Monte Carlo):** 2.68152

## 3. Оцінка похибки
Для оцінки точності методу було проведено порівняння отриманого результату з аналітичним:
* **Абсолютна похибка:** $|2.68152 - 2.66667| \approx 0.01485$
* **Відносна похибка:** $0.5570\%$

## 4. Висновки
1.  **Точність:** Метод Монте-Карло показав високу точність із похибкою менше 1%. Це підтверджує правильність програмної реалізації алгоритму.
2.  **Співвідношення ресурсів:** Хоча результат є наближеним, він був отриманий дуже швидко. Збільшення кількості точок ($N$) дозволило б ще більше зменшити похибку, згідно із законом великих чисел.
3.  **Застосування:** Отримані результати підтверджують, що метод Монте-Карло є ефективним інструментом для чисельного інтегрування, особливо в випадках, коли аналітичне знаходження первісної є складним або неможливим.
