# Порівняльний аналіз результатів обчислення інтеграла

## 1. Аналітичний розрахунок
Математичне значення визначеного інтеграла функції $f(x) = x^2$ на проміжку $[0, 2]$ обчислюється наступним чином:

$$
\int_{0}^{2} x^2 \,dx = \left[ \frac{x^3}{3} \right]_{0}^{2} = \frac{2^3}{3} - 0 = \frac{8}{3} \approx 2.666667
$$

Програмна перевірка за допомогою функції `scipy.integrate.quad` підтверджує цей результат з високою точністю (абсолютна похибка наближається до машинний нуль).

## 2. Результат методу Монте-Карло
При використанні алгоритму Монте-Карло з кількістю випадкових точок *N = 100000*, отриманий результат наближається до аналітичного ($\approx 2.666667$). 

Варто зазначити, що результат змінюється при кожному запуску програми через імовірнісну природу методу, але коливається навколо істинного значення.

## 3. Висновки

* **Точність:** Метод Монте-Карло дає достатньо точне наближення значення інтеграла. Точність результату прямо пропорційна кількості точок $N$ (згідно із законом великих чисел, зі збільшенням вибірки похибка зменшується).
* **Ефективність:** Для простих одновимірних функцій (як у цьому завданні) аналітичні методи або функція `quad` є значно швидшими та точнішими. Метод Монте-Карло потребує більше обчислювальних ресурсів для досягнення високої точності.
* **Застосування:** Хоча для даної задачі метод є менш ефективним за аналітичний, він продемонстрував свою працездатність. Головна перевага методу Монте-Карло розкривається при роботі зі складними функціями, багатовимірними інтегралами або стохастичними процесами, де аналітичне рішення є неможливим або надто складним.
